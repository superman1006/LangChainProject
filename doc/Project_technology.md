## 技术栈

| 技术 | 版本 | 用途 |
|---|---|---|
| Spring Boot | 3.4.3 | 应用框架 |
| Google ADK | 0.5.0 | 智能体编排引擎（LlmAgent、LoopAgent、ParallelAgent、SequentialAgent、InMemoryRunner） |
| Spring AI | 1.1.0-M3 | ChatModel 抽象、OpenAI API 集成、MCP 客户端 |
| LangChain4j | 1.4.0 | 可选 LLM 框架（测试类中有对比示例） |
| xfg-wrench-design-framework | 3.0.0 | 策略树（责任链）设计模式框架 |
| MyBatis | 3.0.4 | ORM（按需启用） |
| MySQL | 8.x | 数据库 |
| Guava | 32.1.3 | 本地缓存、工具库 |
| RxJava 3 | — | 流式事件处理（`Flowable<Event>`） |
